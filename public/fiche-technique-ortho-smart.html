Fiche technique – Ortho Smart

Plateforme d’exercices et d’évaluations orthophoniques (arabe), ludique et progressive.

Objectif & Public

  • Objectif : exercices interactifs et évaluations orthophoniques.
  • Public : orthophonistes, parents / enfants.

Pile technique

  • Frontend : React 18, TypeScript, Vite, React Router v6.
  • Data/UI : @tanstack/react-query, Tailwind CSS, shadcn/ui, Lucide Icons.
  • Audio : Web Speech API (speechSynthesis ar-SA), Web Audio (applaudissements).
  • I18n/RTL : interface arabe, mise en page RTL.

Routes clés

  • /attention-exercises, /perceptual-exercises, /memory-exercises
  • /diagnostic-tests, /diagnostic-response-sheet
  • /theory-of-mind, /theory-of-mind-games
  • /portage-report, /age-calculator
  • Inscription / tableaux de bord : /parent-*, /specialist-*

Authentification & Accès

  • Protection des pages via ProtectedRoute (filtrage par type d’utilisateur).

Exercices

  • Attention :
    • Attention soutenue (attraper l’étoile rouge).
    • Attention sélective (trouver les objets jaunes).
    • Attention conjointe (identifier le locuteur et animation de l’enveloppe vers le destinataire).
  • Perception visuelle :
    • Facile : trouver l’intrus.
    • Moyen : correspondance de formes (cercle, carré, triangle) par glisser‑déposer.
    • Difficile : choisir l’ombre correcte.
  • Autres modules : groupe implicite, acquisitions pré‑basiques, formulaire patient, outil âge, théorie de l’esprit, rapport Portage.

UI & Performance

  • Composants shadcn/ui (Button, Card, Badge, Progress, etc.), styles utilitaires Tailwind.
  • Build Vite, animations légères, pages découpées.

Limitations

  • TTS dépendant du navigateur ; DnD HTML5 ; pas de backend persistant par défaut.
  • Déploiement : configurations Netlify et serveur Node disponibles si besoin.
